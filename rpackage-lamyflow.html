8 Rpackage: lamyflow

8.1 Why building this Rpackage?

We’re almost at the end of this portfolio, and we’ve seen a lot of datasets, codes, functions etc. To reproduce this portfolio alot easier I’ve created this Rpackage called lamyflow.

8.2 Installation

The package “lamyflow” needs to be installed as following:

install.packages("devtools")
devtools::install_github("lamy-m/lamyflow", build_vignettes = TRUE, force = TRUE)

After installation the package can be called as following:

library(lamyflow)

8.3 vignette

To know more about the package, go the vignette with the following code:

browseVignettes("lamyflow")

8.5 Functions

This package contains four functions.

  • Celegans_norm(): Normalizing Celegans data
  • csv_rds(): Saves datafile to .csv and .rds file
  • datainfo(): Gets info about a dataset
  • q_value(): calculates Q-value

To know more about the functions, use ?

?Celegans_norm()
